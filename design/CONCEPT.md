# DailyFinance Pro Next.js concept

Generated with the built-in image generation tool as a `ui-mockup` and used as the implementation specification.

## Final prompt

Redesign the existing DailyFinance Pro chit fund dashboard as a complete 1536×1024 production-ready Next.js SaaS screen while preserving its information architecture. Design for owners, managers, accountants, and collection agents at Indian chit fund companies. Include a persistent left sidebar with company and role context; Dashboard, Customers, Chit Groups, Collections, Reports, Team & Roles, and Settings navigation; a header with date, active company, role, notifications, and profile; collection overview figures; portfolio KPIs; a readable seven-day collection chart; today's target; a table-style collection ledger; agent performance; and recent activity. Make “Record collection” the primary action and “Add customer” secondary. Use a professional financial operations-console direction: deep navy and blue, emerald success accents, true-white surfaces, very light cool blue-gray background, crisp geometric/humanist sans typography, subtle dividers, soft shadows, 12–16px radii, and consistent outline icons. Keep all real UI text and controls code-native and practical for an accessible responsive App Router implementation. Avoid a marketing hero, bento-card overload, excessive pills or gradients, glassmorphism, neon, decorative imagery, fake unrelated metrics, unreadable text, and extra product areas.

## Design lock

- Native reference viewport: 1536×1024
- Primary colors: deep navy sidebar, blue action/selection, emerald successful collections, amber due state, red overdue state
- Container model: open app shell, overview band, compact metric panels, two-column analytics row, table ledger, bottom operational panels
- Typography: Sora for display, Manrope for UI and data
